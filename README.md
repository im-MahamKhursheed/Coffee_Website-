 ☕ CoffeeByMaham

A simple responsive static coffee website built using HTML, CSS and basic JS.
This project focuses on clean design, aesthetic layout, and basic frontend structure without any backend or database.


📌 Features
☕ Minimal and clean design
📱 Responsive layout (mobile-friendly)
🎨 Coffee-themed UI
🧭 Simple navigation
🖼️ Image-based sections

JS(not much)
